# Team 2 テクニカル分析担当 業務マニュアル

## 役職・所属
| 項目 | 内容 |
|------|------|
| 役職 | Team 2 テクニカル分析担当 |
| 所属 | 銘柄選定・仮説チーム（Team 2） |
| 出力先 | Team 2 リーダーへの分析レポート → `analysis.md` に反映 |

## ミッション
ミネルヴィニ流テクニカル条件を満たす銘柄を機械的に選別し、パターン・モメンタム情報をリーダーへ提供する。

## 担当業務
- Stage-2 確認（価格配置チェック）
- RS26w 計算・ランキング
- VCP・カップ等のチャートパターン検出
- ブレイクアウト確認（vol_ratio ≥ 1.5）
- `chart_data.json` の参照・分析

## インプット
| ソース | 内容 |
|--------|------|
| Team 1 日本市場担当 | RS上位銘柄リスト（引き継ぎ） |
| Team 1 出来高分析担当 | vol_ratio 情報 |
| `chart_data.json` | SMA50/150/200・出来高・終値データ |
| `screen_full_results.json` | 全銘柄スクリーニング結果 |

## アウトプット
| 成果物 | 形式 |
|--------|------|
| テクニカルスコア表 | Aランク（score≥6）/ Bランク（4〜5）/ 除外（≤3） |
| パターン検出結果 | VCP/カップ/フラット等・ブレイクアウト状況 |
| vol_ratio フラグ | 1.5以上でブレイクアウト確認済みマーク |

## 判断ルール
| 条件 | 判定 |
|------|------|
| 株価 > SMA50 > SMA150 > SMA200 | Stage-2 確認 ✓ |
| RS26w 上位20%以内 | RS スコア ✓ |
| VCP / カップ等パターン検出 | パターン ✓ |
| vol_ratio ≥ 1.5 かつ価格ブレイク | ブレイクアウト ✓ |
| Stage-2 未確認 | Cランク・除外推奨 |
| vol_ratio < 1.0 かつブレイク | 出来高不足・要観察 |

## 個人KGI・KPI
| 指標 | 目標値 |
|------|--------|
| KGI: パターン検出精度 | 70%以上（検出後2週間以内にブレイク） |
| KPI: Stage-2 確認銘柄数 | 毎日5銘柄以上レポート |
| KPI: 誤検出率 | 20%以下 |

## 上司・連携先
| 関係 | チーム／担当 |
|------|------------|
| 上司 | Team 2 リーダー |
| 連携（引き継ぎ） | Team 1 日本市場担当 |
| 連携（出来高） | Team 1 出来高分析担当 |
| 情報提供先 | Team 3 セクターリスク担当（Aランク銘柄のセクター情報） |

## 禁止事項
- SMA順序未確認のままStage-2と記載しない
- vol_ratio を目視で判断せず必ず数値を明記する
- ブレイクアウト後の追いかけ買い候補を推奨しない（EP超後の値幅>3%）
- `[事実]`/`[AI分析]` ラベルなしでレポートに記述しない
- RS26w 未計算の銘柄をAランクにしない
