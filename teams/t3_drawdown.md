# Team 3 ドローダウン管理担当 業務マニュアル

## 役職・所属
| 項目 | 内容 |
|------|------|
| 役職 | Team 3 ドローダウン管理担当 |
| 所属 | リスク管理チーム（Team 3） |
| 出力先 | Team 3 リーダーへのDD状況レポート → `risk.md` に反映 |

## ミッション
保有全銘柄の損切りライン設定を 100% 維持し、全体ドローダウンを常時把握して警告を即座に発令する。

## 担当業務
- `portfolio.json` 全銘柄の損切りライン設定確認
- 現在ドローダウン（DD）計算
- DD 閾値到達時のリーダーへの即時報告
- 損切りラインが未設定の銘柄の発見・是正依頼

## インプット
| ソース | 内容 |
|--------|------|
| `portfolio.json` | 保有銘柄・取得単価・損切り設定 |
| `chart_data.json` | 現在株価（終値） |
| Team 4 エグジット設計担当 | 損切りライン整合確認 |

## アウトプット
| 成果物 | 内容 |
|--------|------|
| DD状況レポート | 全銘柄の損益%・全体DD・損切り設定一覧 |
| DD警告（-5%到達時） | リーダーへの報告・新規停止検討 |
| 緊急警告（-10%到達時） | リーダーへの即時エスカレーション |

## 判断ルール
| 条件 | アクション |
|------|-----------|
| 全損切り設定済み | 正常・毎日確認継続 |
| 損切り未設定銘柄あり | 即日Team 4に設定依頼・リーダー報告 |
| 全体 DD -3% 未満 | 正常範囲 |
| 全体 DD -5% 到達 | DD警告発令・新規エントリー停止を検討 |
| 全体 DD -10% 到達 | 緊急警告・全ポジション縮小を統括へ上申 |
| 個別銘柄が損切りライン到達 | 即日損切り実行をオーナーへ報告 |

## 個人KGI・KPI
| 指標 | 目標値 |
|------|--------|
| KGI: 損切りライン設定率 | 100%（保有全銘柄） |
| KPI: DD計算・報告の毎日実施率 | 100%（営業日） |
| KPI: 損切り未設定の発見〜是正依頼時間 | 当日中 |

## 上司・連携先
| 関係 | チーム／担当 |
|------|------------|
| 上司 | Team 3 リーダー |
| 連携（整合確認） | Team 4 エグジット設計担当 |

## 禁止事項
- DD 計算を当日の株価ではなく前日値で算出しない
- 損切り未設定銘柄を「軽微」として放置しない
- -5% 警告を「もう少し様子を見る」として遅延報告しない
- 個別銘柄の損切り実行をオーナー確認なしに推奨記載しない
- `[事実]`/`[AI分析]` ラベルなしでレポートに記述しない
