# イベント管理チーム ナレッジベース

## イベントタイプ定義
- earnings: 企業決算発表（日本・米国・その他）
- economic_indicator: 経済指標発表（CPI・GDP・雇用統計・PPI・機械受注など）
- central_bank: 中央銀行イベント（日銀会合・FOMC・ECBなど、ブラックアウト期間含む）
- ipo: 新規上場
- other: その他（IMF会合・G7・市場休場など）

## 重要度定義
- critical: 市場フェーズ判定に直結（TSMC決算・日銀会合・FOMC）
- high: セクターへの大きな影響（主要企業決算・主要経済指標）
- medium: 参考情報（中小企業決算・マイナー指標）
- low: 記録のみ

## 重複排除ルール
- id フィールドで一意性を管理
- id命名規則: {ticker/name}-{type}-{year} 例: tsmc-earnings-2026q1
- 同一イベントの日付変更は上書き（削除→追加ではなくdate更新）

## 関連銘柄コードの記載基準
- related_codes: 決算やイベントの影響を直接受ける日本株コード（追跡中・ウォッチ中銘柄を優先）
- TSMC決算 → 6857(アドバンテスト)・8035(東京エレクトロン)・6594(日本電産)
- 日銀利上げ → 3003(ヒューリック)・8306(三菱UFJ)・8316(三井住友)
